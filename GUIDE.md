# راهنمای استفاده

## فایل‌های پروژه

پروژه شامل فایل‌های زیر است:

- **index.html** - صفحه اصلی سایت
- **style.css** - فایل استایل با تم آبی-بنفش
- **app.js** - فایل جاوااسکریپت برای خواندن و نمایش محتوای مارکداون
- **readme.md** - فایل مارکداون حاوی لینک‌های دانلود ویدیوها
- **fonts/** - پوشه حاوی فایل‌های فونت وزیر

## نحوه استفاده

### افزودن ویدیوهای جدید

برای افزودن ویدیوهای جدید، فایل `readme.md` را ویرایش کنید و به فرمت زیر عمل کنید:

```markdown
## نام تمرین

### عنوان ویدیو
- **توضیحات:** توضیحات مربوط به ویدیو
- [دانلود](لینک-دانلود-ویدیو)
```

### مثال:

```markdown
## تمرین هفتم - کار با فایل‌ها

### سوال ۱: خواندن از فایل
- **توضیحات:** نحوه باز کردن و خواندن محتوای فایل‌های متنی
- [دانلود](https://example.com/video20.mp4)

### سوال ۲: نوشتن در فایل
- **توضیحات:** نوشتن داده در فایل‌ها و کار با حالت‌های مختلف
- [دانلود](https://example.com/video21.mp4)
```

## نمایش سایت

### روش ۱: استفاده از فایل START.bat (ساده‌تر)
فایل `START.bat` را دابل‌کلیک کنید. سرور محلی راه‌اندازی می‌شود و می‌توانید از آدرس زیر سایت را مشاهده کنید:
```
http://localhost:8000
```

### روش ۲: استفاده دستی از Python
در پوشه پروژه، دستور زیر را اجرا کنید:
```bash
python -m http.server 8000
```
سپس مرورگر را باز کنید و به آدرس `http://localhost:8000` بروید.

**نکته:** برای متوقف کردن سرور، `Ctrl+C` را فشار دهید.
